# Supported Proof Systems

Fermah is designed to accommodate any proving system. These are the ones we currently support:

- [Jolt by a16z](https://jolt.a16zcrypto.com/)
- [Groth16 by Jens Groth](https://www.zeroknowledgeblog.com/index.php/groth16)
- [The RISC Zero zkVM](https://dev.risczero.com/api/)
- [Valida by Lita](https://lita.gitbook.io/lita-documentation)
- [Stwo by Starkware](https://github.com/starkware-libs/stwo)
- [SP1 by Succinct Labs](https://docs.succinct.xyz/)

Very soon, we will also support:

- [The Nexus 2.0 zkVM](https://docs.nexus.xyz/)
- [Stone by Starkware](https://github.com/starkware-libs/stone-prover)
- [The Aztec Proof System](https://docs.aztec.network/)

This initial support allows developers to integrate these systems quickly and leverage Fermah's capabilities without additional setup.

We plan to support all major proving systems.
